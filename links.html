Lokale Entwicklung – Schnellstart Kunden-Website

# Kunden-Website cd C:\Users\mail\klartext-liebe npm install # einmalig npm run dev # startet auf http://localhost:3000 npm run build # baut statische Seite → out/ # Änderung live bringen git add -A git commit -m "Kurze Beschreibung der Änderung" git push # → Cloudflare baut & deployed automatisch (~2 Min.)
⚠️

Wichtige Hinweise & Stolperfallen

  • Nie npm 11 für package-lock.json nutzen → Cloudflare CI nutzt npm 10.9.2. Fix: npx npm@10.9.2 install --package-lock-only
  • Vor jedem Push lokal npm run build testen (Ordner out/ wird erzeugt)
  • Bilder: images.unoptimized: true in next.config.mjs nötig (statischer Export)
  • Windows curl schlägt fehl → stattdessen PowerShell Invoke-WebRequest nutzen
  • EBUSY-Fehler beim Build → einfach nochmal bauen (Antivirus blockiert kurz)
  • Push auf main → automatisch live – also nur bewusst pushen, lokal vorher testen
  • Markenfarben: Türkis #3FB9C7 · Blau #3B82C4 · Violett #7C6BC4 · Creme #FAF7F2
