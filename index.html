Tech-Bausteine

⚛️ Next.js 14 (App Router)

Grundgerüst der Website. Statischer Export. Baut alle Seiten, verwaltet URLs, optimiert Ladezeiten. TypeScript + Tailwind CSS.

☁️ Cloudflare Workers / Pages

Hosting und Auto-Deploy. Fertige Website wird weltweit schnell ausgeliefert. Kostenlos bis zu sehr hohen Zugriffszahlen.

🗄️ Supabase (geplant, Phase 4)

PostgreSQL-Datenbank als Service. Auth (E-Mail/Passwort + Magic Link), Datenspeicherung für Waitlist, Kontaktanfragen, Nutzerprofile, Community.

💳 Stripe (geplant, Phase 3)

Zahlungen für Coaching-Pakete. Payment Links funktionieren ohne eigenes Backend. Webhook → Supabase für automatische Freischaltung.

🤖 Claude AI (geplant, Phase 7)

Anthropic Claude für die KI-Persönlichkeitsanalyse. Fragebogen-Antworten werden tiefgehend interpretiert und als persönliches Profil gespeichert.

Tech-Stack

Frontend

  • Next.js 14 (App Router)
  • TypeScript + Tailwind CSS
  • Cloudflare Workers

Backend / Services

  • Supabase (Auth + DB) → Phase 4
  • Stripe (Payments) → Phase 3
  • Claude API (KI) → Phase 7
